# Guía de Contribución

¡Gracias por tu interés en contribuir a este proyecto (Mi repositorio de perfil de GitHub)!

## Cómo contribuir

Este repositorio sirve principalmente como mi portafolio personal y presentación en GitHub. Sin embargo, si deseas contribuir (por ejemplo, sugiriendo mejoras al formato, reportando un link roto o recomendando una mejor estructura), ¡tus aportes son bienvenidos!

1. Haz un **Fork** del repositorio.
2. Crea una **nueva rama** (`git checkout -b fix/mejorar-lectura`).
3. Haz tus cambios y asegúrate de correr el linter con `make lint`.
4. Haz **Commit** de tus cambios usando Conventional Commits (ej. `docs: arreglar link roto en la sección habilidades`).
5. Haz **Push** a la rama (`git push origin fix/mejorar-lectura`).
6. Abre un **Pull Request** detallando qué se cambió y por qué.

## Desarrollo Local

Este repositorio asume un entorno compatible con Linux/Fedora. Para más detalles, consulta la sección "Cómo ejecutar localmente (Fedora Linux)" dentro del archivo `README.md` o usa el `Makefile` incluido para interactuar con las herramientas del proyecto.
